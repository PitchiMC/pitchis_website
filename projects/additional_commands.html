Enhance your server experience with essential tools. From teleportation to utility shortcuts, all in one lightweight package.

Download on Modrinth Get Support

Better Server Control

Stop struggling with vanilla limitations. Additional Commands adds the most requested features for server owners and players alike, without the bloat of massive "all-in-one" plugins.


Whether you need a quick way to heal, fly, or manage player locations, this plugin provides a clean and intuitive command structure that fits perfectly into any survival or creative server.

Included Features

  • Essentials: /heal, /feed, and /fly support.
  • Teleportation: Set homes and warp across the world.
  • Utility: Quick access to crafting tables and enderchests.
  • Permissions: Fully compatible with LuckPerms.
  • Optimized: No lag, just pure functionality.

Quick Setup

permissions.txt 
# Grant access to all basic commands
additionalcommands.user.*

# Grant access to admin features
additionalcommands.admin.*
