Wireless Redstone

The ultimate cable-free solution. No trails, no repeaters – just Lightning Rods and a frequency book.

Download on Modrinth Support Discord

Cables are History

Every redstone engineer knows the pain of hiding long lines of dust and repeaters. Wireless Redstone uses Lightning Rods as powerful antennas to transmit signals through the air.


By simply using a book to set frequencies, you can connect complex machines across chunks without a single block of redstone dust in between. It’s clean, it’s fast, and it looks vanilla.

Core Features

  • Vanilla Style: Uses Lightning Rods as antennas.
  • Infinite Channels: Create unique frequencies via Book & Quill.
  • Visual Feedback: Rods glow when receiving a signal.
  • Oxidation Support: Works with all Rod variants (Waxed, Oxidized, etc.).
  • Ultra Light: Zero lag, optimized for high-speed signal updates.

Configuration

config.yml 
# Wireless Redstone Data Storage
# This file stores your registered senders and receivers.
# Do not edit manually unless you know what you are doing.

data:
  '100':
    senders:
    - world,10,64,10
    receivers:
    - world,20,64,20
